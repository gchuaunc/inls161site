Angular site for INLS 161 at UNC-CH | created by Gabrian Chua


Welcome to oneSIXone, a site all about me, created for INLS 161 at the University of North Carolina at Chapel Hill. INLS stands for Information and Library Sciences, but it is so much more than that! Feel free to explore this site and learn more about me.

Features

Some special features I've included in this site include:

  • Angular. Angular is ia modern web framework that allows highly flexible and efficient website design and creation. I like it especially for its ability to create modular "components" which you can re-use all across your site. For example, the navbar component and the footer component is re-usable, meaning one edit to that component propagates across the site (no need to copy and paste across all pages anymore!).
  • Responsive design. I used Angular's BreakpointObserver to observe the size of your screen and change the site's style accordingly. If you are on a sufficiently small screen, the site will switch to "mobile" layout: the navbar is minimized into a hamburger menu and the content takes up 100% of the width. Otherwise, the site takes on the "desktop" layout: the navbar is expanded and the width of the content is minimized to at most 800 px, or 60% of the view width, whichever is smaller.
  • Angular Material. Angular Material is a popular library by Google that implements Material Design into your site and makes many prewritten and configurable components available for use, like buttons, drawers, drop-downs, and more. You can find the list of components at the documentation page.
  • Angular Animations I enabled and wrote a few simple animations to make the UI a little bit smoother. Examples include the drop down menu in the navbar and the hamburger menu in mobile mode.
  • Image carousel component. I custom-wrote a simple image carousel to showcase the pictures I wanted to show in a compact and elegant way.
