Interesting Facts

Some interesting facts about me:

  • My main laptop, which I use every day for school, runs Linux Mint 21.2! Ever since it was dropped for Windows 11 support, I explored using Linux as the main operating system alongside Windows 10. I already had a bit of experience with Linux in the form of Debian, since I had owned a Raspberry Pi for a few years. Eventually, I needed to reclaim some space and realized I hadn't booted into Windows in the last 2 months. So, I decided to clear the Windows partition and reallocate that space for Linux's ext4! I found that Linux Mint can do most everything I need it to, except for a few choice apps that are only available for Windows (in which case, I have a QEMU Virtual Machine with Windows 11).
  • Hate ads? Me too. I run a Pi-Hole node on my home network, which is a DNS-level ad blocker. The way it works you install the software onto a Linux computer (in my case, a small Debian Linux headless laptop server), which starts up a DNS server that stands in between your network's devices and a real DNS provider like Google or Cloudflare. Then, when the Pi-Hole gets a request for a known ad-domain like doubleclick.net, it rejects the request and sends back a garbage IP address like 0.0.0.0 so that the client cannot load the ad. Not only does this remove ads, but it also makes your network faster because it doesn't need to spend bandwidth loading the network-heavy images and videos!
